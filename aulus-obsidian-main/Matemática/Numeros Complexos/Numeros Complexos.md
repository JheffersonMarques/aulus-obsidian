> Numeros complexos foram criados para resolver um problema que se tinha cuja a resolução não existia pela formula padrão (Bhaskara).


### Exercicio 1
```
{2x + y + 2 = 0, y = x² - 5x}

-b +- Math.sqrt(b²*4*a*c)/2*a

3 +- Math.sqrt(-3²*-4*1*2)/2

x1 = 2
x2 = 1
```
